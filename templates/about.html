About Stat Lemon

Stat Lemon is an NBA analytics sandbox that provides predictive insights for fantasy basketball players using SARIMA time series modeling.

Our Methodology

We use advanced time series analysis to identify patterns in player performance data and generate predictions for upcoming games. Our approach includes:

  • SARIMA (Seasonal AutoRegressive Integrated Moving Average) modeling
  • Player-specific performance pattern analysis
  • Advanced metrics beyond traditional box scores
Current Features
  • Player performance forecasting
  • Head-to-head player comparisons
  • Lineup builder with team analysis
